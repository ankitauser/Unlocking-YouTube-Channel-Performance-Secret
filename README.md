# Unlocking YouTube Channel Performance Secrets

##  Project Overview
This project analyzes YouTube video analytics to uncover engagement trends, optimize content strategies, and predict revenue growth.  
Using Python and machine learning, the analysis highlights how views, subscribers, and engagement metrics drive monetization.

##  Tools & Technologies
- Python (pandas, NumPy, scikit-learn, seaborn, matplotlib)
- Jupyter Notebook / VS Code
- Machine Learning (Random Forest Regressor)
- Data Visualization

##  Dataset
The dataset includes:
- **Video Details**: Duration, publish time, day of week  
- **Revenue Metrics**: Estimated revenue, CPM, ad impressions  
- **Engagement Metrics**: Views, likes, shares, comments, CTR  
- **Audience Data**: Subscribers, unique viewers, returning viewers  

*(Dataset source: Provided by Unified Mentor)*

## Workflow
1. Import libraries and load dataset  
2. Data cleaning and preprocessing  
3. Exploratory Data Analysis (EDA)  
   - Correlation heatmaps  
   - Revenue vs Views scatterplots  
   - Top-performing videos analysis  
4. Feature Engineering  
   - Revenue per view  
   - Engagement rate  
5. Predictive Modeling  
   - Train/test split  
   - Random Forest regression  
   - Model evaluation (MSE, R²)  
6. Insights & Recommendations  
   - Key drivers of revenue  
   - Underperforming areas  
   - Feature importance visualization  

## Key Insights
- Views and engagement rate are the strongest predictors of revenue.  
- High CPM and monetized playbacks significantly boost channel earnings.  
- Content published on certain days shows higher retention and subscriber growth.  

##  Deliverables
- Jupyter Notebook with full analysis  
- Visualizations (heatmaps, scatterplots, bar charts)
